Calculates a simplex noise value for a 3d voxel or a 2d pixel and 1d time.

To produce Cloud/Terrain like textures add several octaves. By holding one or two dimensions fixed you can extract lower dimensional noise.

Ref: Simplex-Noise-Demystified (read this) Ref: Simplex Noise by Ken Perlin (not read) Ref: "Improving Noise" by Ken Perlin (16 instead of 12 gradients) I just implemented it adding permutation through LFSR and with calculation of an edge-direction lookup index for a table instead.

Parameters
seedan octave unique seed value in [0,255].
Returns
float value in about [-1,+1].
