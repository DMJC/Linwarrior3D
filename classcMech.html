virtual

Called to constrain the given world position particle (with radius) to the object boundary hull ie.

place the worldpos to the nearest boundary hull position if the worldpos is inside the object's hull. localpos delivers in case of impact the local position relative to the object of the corrected worldpos. @worldpos input location vector in world coordinates. @radius radius of the checked particle in world dimensions. @localpos output adjusted location vector.

Returns
the intrusion depth.
