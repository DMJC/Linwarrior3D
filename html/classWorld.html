◆ fragObject()

void fragObject ( Entity *  object)

Removes object from active world (mObjects & mIndex), adds object to Corpse-List, adds rRole::FRAGGED role to object, finally object->onFragged() is called.

Note that FRAGGED != DEAD objects: Dead objects are still part of the world but fragged objects are removed from the perceptible world. Fragged == removed and scheduled for deletion.
