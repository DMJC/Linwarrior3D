Detailed Description

Textured Console Font out of a box.

For opengl use glUploadFont() and then glprintf("Hello World"); used in Console class.

See method definitions for further comment and information. See end of file for a testdriver/sample of use.

◆ glUploadFont()

static unsigned int* glUploadFont ( )
inlinestatic

Use this function to generate and upload GL textures for ascii characters { start_ascii+0, start_ascii+1, ..., start_ascii+howmany-1 }, where start_ascii == 0 and howmany = 256.

The corresponding GL bind ids are stored back into gFontBind[0], gFontBindv[1], ..., gFontBind[howmany-1]. gFontBind is a user defined pointer - memory is allocated by this function and it is user-responsibility to free and set to NULL when not used anymore.
